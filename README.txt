Al Ajamy Food - Invoice Tracker (Static Site)

Files:
- index.html
- logo.png
- favicon.ico + favicon PNGs

How to use:
1) Upload all files in this folder to your hosting/public directory (same folder).
2) Open index.html in the browser.

Notes:
- This version is client-side only (no database). Use the "Export JSON" button to download invoice data.
- To connect to a database, add an API endpoint and replace the export action with a POST request.
